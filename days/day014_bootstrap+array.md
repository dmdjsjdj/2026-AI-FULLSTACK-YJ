```
1. __container__
   - 전체 레이아웃을 감싸는 박스

   - `container`(고정 폭) / `container-fluid`(가변 폭)

2. __row___
   - `container` 안에서 줄(행)을 만드는 역할   

3. __col__
   - `row` 안에서 칸(열)을 만드는 역할  
   - 기본 단위는 **12칸**
```
